## **Project Title**

Task Track. It's a full-stack project.

## **Project setup**

- Clone the project.
- run npm install.
- Create a (.env) file in the root folder.
- Add value in (.env) file
  - DB_USER
  - DB_PASS
  - JWT_SECRET
- run command ( npm run dev or yarn dev)
